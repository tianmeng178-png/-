## 🔄 协作流程

### 每日站会（建议）
1. **进度同步**（5分钟）
   - 昨天完成了什么？
   - 今天计划做什么？
   - 有什么阻塞问题？

2. **接口对齐**（10分钟）
   - API接口变更通知
   - 数据结构更新同步
   - 依赖关系确认

### 代码集成规范
```bash
# 分支命名
feature/A-参数面板优化
feature/B-websocket优化
feature/C-docker配置

# 提交信息格式
feat(A): 添加参数预设模板功能
fix(B): 修复WebSocket重连问题
docs(C): 更新OpenFOAM安装文档
```

### 接口契约（API Contract）
```typescript
// 仿真启动请求
POST /api/simulation/start
Request: {
  parameters: MicrochannelParameters
}
Response: {
  simulation_id: string,
  status: "started"
}

// WebSocket消息格式
{
  type: "progress" | "completed" | "error",
  data: {
    progress: number,      // 0-100
    current_step: string,
    log_messages: string[],
    performance_metrics?: PerformanceMetrics
  }
}
```

## 🆘 问题解决机制

1. **先自查**（15分钟）
